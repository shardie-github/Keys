# Quick Start: GitHub Repository Templates

Get a new project started in 2 minutes.

---

## Step 1: Choose Template

Select the template that matches your project type:
- SaaS Starter
- API Starter
- CLI Starter
- Full-Stack Starter
- Library Starter

---

## Step 2: Use Template

1. Go to the template repository on GitHub
2. Click "Use this template"
3. Create your new repository

---

## Step 3: Clone and Setup

```bash
git clone <your-new-repo-url>
cd <your-project-name>
npm install  # or follow template-specific setup
```

---

## Step 4: Customize

- Update README.md
- Change package.json name
- Customize configuration files
- Add your code

---

## Next Steps

- Read template-specific README for detailed setup
- Follow template documentation
- Start building your project

---

**That's it!** You have a new project with best practices structure.

# Quick Start: GitHub Issue Management Patterns

Get issue automation working in 2 minutes.

---

## Step 1: Copy Workflows

```bash
cp .github/workflows/* /path/to/your/repo/.github/workflows/
```

---

## Step 2: Configure

Edit workflows to match your team structure and labels.

---

## Step 3: Test

Create a test issue or PR to verify workflows run.

---

**That's it!** Issue management is now automated.

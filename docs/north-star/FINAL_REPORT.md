## Summary

KEYS has been successfully realigned around a single, durable north star:

**KEYS IS NOT AN AI TOOL.  
KEYS IS THE KEYRING TO MODERN TOOLS.**

This pivot clarifies KEYS' identity as a marketplace of structured assets (notebooks, prompts, workflows, playbooks) that unlock practical, repeatable, commercial capability from external tools without competing with them.

---

## The Clarified Identity of Keys

KEYS is the keyring to modern tools. Just as a physical keyring holds keys that unlock doors, KEYS holds structured assets that unlock capability in digital tools like Cursor, Jupyter, GitHub, Stripe, Supabase, and more.

**The tool provides the power. The key provides the leverage.**

KEYS positions itself above the tool wars, providing keys that unlock tools regardless of which tools users choose. This positioning is durable, calm, and future-proof. If a new AI tool appears tomorrow, KEYS still makes perfect sense: "Here are keys to unlock that tool."

## What Existing Work Is Now Better Explained

### Cursor Keys (Previously "Templates")
**Before**: Described as "Scaffold Prompt Templates" - a system for customizing prompts.

**Now**: Framed as "Cursor Keys" - structured prompt packs and Composer instructions that unlock advanced workflows in Cursor. The positioning is clear: Cursor is the tool, Cursor Keys unlock it.

### Jupyter Keys (Previously "Notebook Pack Library")
**Before**: Described as "Notebook Pack Library" - a marketplace of notebook packs.

**Now**: Framed as "Jupyter Keys" - notebook packs that unlock data science workflows and analysis patterns in Jupyter. The positioning is clear: Jupyter is the tool, Jupyter Keys unlock it.

### Marketplace Infrastructure
**Before**: Generic marketplace language without a unifying story.

**Now**: All marketplace assets are framed as "keys" that unlock capability in specific tools. The taxonomy (by tool, by outcome, by maturity) provides a clear classification system.

### Product Principles
**Before**: No explicit guardrails preventing identity drift.

**Now**: Ten non-negotiable principles ensure KEYS never competes with tools, never hides execution paths, and always produces tangible outputs.

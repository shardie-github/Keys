Getting Started

Welcome to the KEYS documentation. This guide will help you integrate our standard visual identity across all your static pages, set up your development environment, and configure the core library.

Installation #

To get started with KEYS, you need to install the core package using your preferred package manager. We support npm, yarn, and pnpm.

Terminal
$ npm install @keys-ui/core @keys-ui/icons
info
Peer Dependencies: Make sure you have React 18 or later installed in your project as KEYS relies on the new concurrent features.

Basic Usage #

Import the styles at the root of your application, usually in main.tsx or App.tsx. Then wrap your application with the KeysProvider to enable the theme context.

App.tsx
import { KeysProvider, Button } from '@keys-ui/core'; import '@keys-ui/core/dist/style.css'; export default function App() { return ( <KeysProvider> <div className="app-container"> <h1>Hello World</h1> <Button variant="primary">Get Started</Button> </div> </KeysProvider> ); }

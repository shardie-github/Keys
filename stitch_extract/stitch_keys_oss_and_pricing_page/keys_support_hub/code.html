Frequently Asked Questions

How do I reset my API keys?

expand_more

You can roll your API keys directly from the dashboard. Go to Settings > API Keys and click "Roll Key". Be aware that this action is immediate and will invalidate your old keys.

What is the difference between the Starter and Pro plan?

expand_more

The Starter plan allows up to 10k requests per month and 3 team members. The Pro plan includes unlimited requests, SSO integration, and priority support response times.

Where can I find the SDKs?

expand_more

We officially support Node.js, Python, and Go. You can find the installation guides and GitHub repositories in our Documentation section.

How do I contact support for billing issues?

expand_more

For sensitive billing inquiries, please use the form above labeled "Contact Support" and select "Billing" from the dropdown, or email us directly at billing@keys.com.

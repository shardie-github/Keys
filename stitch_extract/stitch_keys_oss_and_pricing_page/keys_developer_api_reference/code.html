GET /v1/vaults/{vault_id}

Retrieve a Vault

Returns a vault object if a valid identifier was provided. When retrieving a vault, you'll get the metadata and status of the container, but not the individual secrets within.

Path Parameters

Parameter Type Description
vault_id string Required The unique identifier for the vault container.

Query Parameters

Parameter Type Description
expand array Expand nested objects. Valid values: owner, metadata.
include_tags boolean Whether to include custom tags in the response. Default is false.

Errors

Common error status codes for this endpoint:

404

Not Found

The requested vault ID does not exist in your organization.

401

Unauthorized

Your API key is invalid or has expired.

# Retrieve a vault by ID
curl -X GET \
  https://api.keys.dev/v1/vaults/vlt_47k291 \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json'

Response Body

200 OK 
{
  "id": "vlt_47k291",
  "object": "vault",
  "name": "Production Core",
  "created_at": 1683921000,
  "metadata": {
    "env": "prod",
    "region": "us-east-1"
  },
  "status": "active"
}
